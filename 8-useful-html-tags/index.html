This is the Details disclosure element
This HTML element creates a disclosure widget in which information is visible only when the widget is toggled into an open state. A summary or label must be provided.A disclosure widget is typically presented onscreen using a small triangle that rotates (or twists) to indicate open/closed state, with a label next to the triangle. The contents of thes ummary element are used as the label for the disclosure widget. The contents of the details provide the accessible description for the summary.
